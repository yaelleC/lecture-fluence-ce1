# Prompts d'illustration — Livret de Lecture CE1

17 prompts (1 cover + 4 textes courts + 12 textes principaux) designed to be used with any image generator (DALL-E / ChatGPT, Midjourney, Imagen, Stable Diffusion, Adobe Firefly).

> **Note format :** the 4 Textes courts use **square** images (1:1) because they're inserted with wrap-around text in the booklet's 5×5 cm box. The 12 main texts use landscape if you want full-page illustrations, or square if you want to drop them into wrap-figure boxes too.

## Style commun

All 13 prompts share the same style block so the booklet looks consistent. If you want a different look, replace this one line across all prompts and regenerate:

> **STYLE :** *warm hand-painted watercolor children's book illustration, soft pastel colors, gentle outlines, friendly and inviting, suitable for a 7-9 year old, no text, no words, no letters, no signs*

If you use **Midjourney**, append `--ar 3:2 --style raw` for landscape page illustrations, or `--ar 3:4` for the cover. For **DALL-E / ChatGPT**, just describe the orientation in words ("landscape" / "portrait cover").

**Orientation :** portrait

**Prompt :**

> A cheerful young girl with a big smile sitting cross-legged on a soft rug, holding an open book in her lap. ## Conseils d'utilisation

- **Pour ChatGPT (DALL-E)** : copie-colle un prompt à la fois. Tu peux ajouter "in landscape orientation" si l'image ne sort pas dans le bon format.
- **Pour Midjourney** : ajoute `--ar 3:2` pour les pages et `--ar 3:4` pour la couverture. Tu peux aussi essayer `--style raw` pour un rendu plus illustratif.
- **Si une image te plaît mais pas le visage de l'enfant** : précise dans le prompt "child with [color] hair and [color] eyes" pour ressembler à ta fille.
- **Cohérence** : si tu veux que tous les enfants illustrés se ressemblent (même petite fille à travers le livret), ajoute la même description physique dans chaque prompt avec un enfant.
